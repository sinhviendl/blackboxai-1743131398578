How do I center a div vertically and horizontally?

You can center a div using CSS Flexbox:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

Or with CSS Grid:

.container {
    display: grid;
    place-items: center;
    height: 100vh;
}
